## 🐳 설명
<p align="center">

### 제안 배경
- 교사들의 업무 효율 향상을 통해서 업무로부터 받는 스트레스를 줄이고자 서비스를 개발하기 시작하였습니다.

### 주요 내용
- 사용자(교사)가 학교 생활간에 사용하는 각 일지(학급일지, 업무일지, 상담일지, 관찰일지,할일 목록)를 작성 및 수정하며 관리할 수 있고 각 학기별로 업무를 관리하며 언제든지 아카이브를 통해 조회 할 수 있다.
